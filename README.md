## ⭐ 서비스 개요
눈송이가 마을에 작은 카페를 열었습니다.

카페의 이름은 Snow Cafe.

손님이 많아 도와줄 사람이 필요하다고 합니다.

음료수를 만들어 송이를 도와주세요



## ☕ 게임 방법
메뉴판에 등장하는 음료수를 만드는 게임입니다.

음료수에 들어가는 재료를 드래그해서 넣어주세요.

왼쪽 상단에 있는 버튼을 누르면 레시피를 볼 수 있어요.

총 5개의 스테이지이며 제한 시간은 3분입니다.
